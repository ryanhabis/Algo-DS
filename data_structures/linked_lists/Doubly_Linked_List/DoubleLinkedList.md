# DoublyLinkedList

A simple implementation of a doubly linked list in Java.

## Features

- Add elements to the end of the list
- Print all elements in order

## Usage

```java
DoublyLinkedList list = new DoublyLinkedList();
list.add(10);
list.add(20);
list.add(30);
list.printList(); // Output: 10 20 30
```
